About Me

Go to www.vmware.com and install the workstation trial. Once installed, download the .ISO from the CentOS website. Make sure that is the DVD version not the minimal. From there you can then open the VMWare workstation and create a new “Virtual Machine”. Select the number of cores, ram, hard drive space that you want to dedicate to the virtual machine. I personally would keep it the same. Making sure that in the selection of what operating system that you point is at the ISO that you just downloaded. The Virtual machine will then start, and you will be greeted with a CentOS installation. Make sure that you set the version to the server installation. This is so that you have all the packages install to edit and maintain the CentOS Server Core. Make sure that you set the root password to something that you will remember as you will be using it frequantly. After the install is finished open a terminal and start by installing the apachie server using the command yum -y install httpd. Once that is done you can then start and check the status of the server. This is done by doing the following

  1. Open a terminal by chicking the windows key and typing in Terminal.
  2. Make yourself the root user for that terminal.
  3. Use the command su– to get to root user.
  4. Enter your root users password.
  5. In the terminal enter systemctl start httpd.
  6. Now check the status of the webserver to make sure that you started the server. Do this by typing in systemctl status httpd, this will show you the process and from there you can confirm that the service has started. To check if its running open up a internet browser and type in localhost this will tell the browser that you want to access the web server.
  7. After the webpage loads you can then confirm that the server is running, if the site doesn’t show up you can then disable the firewall by typing systemctl stop firewalld. This will allow for the webpage to show pictures and allows other users to see your website.
  8. Now you can go into the directory and edit the html page. You can do this by opening a terminal and typing in cd/var/www/html in this directory you will find 2 files the one that we want to change is the index file to do that we will type nano index.html in this file we will put our code for out website.
